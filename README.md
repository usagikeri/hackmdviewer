# hackmdviewer
A local server that displays a list of files on hackMD.

# Requirement
* docker

# Install
```
$ git clone https://github.com/usagikeri/hackmdviewer.git
$ sh install.sh
```

# Usage
1. Access localhost:3000, hackMD.
2. Create markdown file.
3. Access localhost:8888, flask web server.
